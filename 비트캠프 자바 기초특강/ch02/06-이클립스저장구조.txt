
<.java vs. .class>

.java 소스파일 : src 폴더
.class 바이트코드 : bin 폴더 (필요하지 않기 때문에 분리해놓음)

<new '자바프로젝트' 만들기>
파일>new>자바프로젝트

<.java 소스파일 만들기>
프로젝트명>new>class(*.java)>모듈,패키지 생략 ...

<이클립스의 컴파일>
'저장'시 자동컴파일 : 오토세이브 추천

<주석문>
소스코드내에 설명하는 부분(실행 안됨)
// 주석문 : 한줄 설명문(코드 on/off 로도 쓰임)
/* ~ */ : 연속범위 여러줄 설명문 (C 에서 유래됨)
Ctrl + / : 해당 블럭의 on/off

<백업과 복구>
작업공간에서의 삭제
1. 일반삭제 : 파일시스템 존재, 작업공간목록에서만 삭제
2. 완전삭제 : 파일시스템에서도 삭제

백업 : 파일>export>general>File System>원하는 대상선택>백업받을곳

복구 : 파일>import>existing project .. > 원하는 대상선택>copy .. 체크
