bin : binary (실행파일)

<자바프로그램 종류>
JDK(Java Development Kit) : 자바프로그래밍 개발시 필요
컴파일러 + API 라이브러리로 이루어짐

컴파일러란?
사람의 자연어를 컴퓨터가 쓰는 기계어로 번역해주는 번역기

라이브러리란?
개발에 필요한 기본 부품들을 모아놓음

JRE(Runtime Environment): 자바 실행 환경
자바 프로그램 실행시 필요

JVM(Virtual Machine): 플랫폼 독립적 실행에 필요


JDK만 설치하면 OK : JDK ⊃ JRE ⊃ JVM (전부 설치됨)

<자바설치여부>
1. 제어판 앱 목록 (제어판 -> 앱 -> JDK 확인)
2. C:\Program Files\Java 폴더 확인
(자바 삭제시 폴더를 삭제하면 안된다.)
3. cmd 에서 java 로 확인

<다운로드 폴더 확인>
탐색기에서 확장자가 항상 보이도록
보기 -> 파일확장명

<환경변수설정>
내PC -> 속성 -> 고급 시스템설정 -> 고급 -> 환경 변수

사용자변수  : 로그인한 사용자한테만 적용
시스템변수 : 
1. 새로 만들기 -> 변수이름 : JAVA_HOME , 변수 값 : (자바 설치 경로 입력)
2. 시스템 변수 Path 편집 -> 새로 만들기 -> %JAVA_HOME%\bin 추가

<설치 확인>
cmd -> java -version 으로 확인

콘솔명령창: 윈 + r > cmd
