IDE(Integrated Development Environment)
통합 개발 환경

자바 IDE 예) 에디트플러스, 이클립스, 인텔리J아이디어, 넷빈즈, ...

<이클립스 특징>
무료
자바 개발자들은 대부분 선호
이클립스 자체도 자바로 개발되고 실행 (즉, 자바를 설치하고 이클립스를 설치해야 한다.)

<이클립스 다운로드>

[이클립스 사이트](https://www.eclipse.org/)

A. 인스톨러에서 직접 설치 (비추)

Download -> Get Eclipse IDE 2020‑12 Download(버전은 회사마다 다름) -> eclipse-inst-jre-win64.exe 설치
-> Eclipse IDE for java Developers(SE 버전) -> 작업공간지정 -> 
(자바 버전에 맞는 이클립스를 사용해야 함)

이클립스는 설치 후 앱 목록에 없음 : 무설치 버전
폴더로 확인 : C:\Users\사용자명\eclipse


B. 압축파일을 수동으로 설치 (추천)

Download -> Download Packages -> 윈도우 버전 압축파일(.zip) 다운로드 받기 ->
압축 풀기 -> C:\Users\사용자명 에 압축해제 -> 작업공간지정

<기본 한글 인코딩 변경>
Windows -> preference -> general-> workspace > text file encoding -> utf-8


<이클립스 삭제>
C:\Users\사용자명\eclipse 삭제
C:\Users\사용자명\eclipse-workspace 삭제
바로가기 아이콘까지 삭제
